## Overview
Implement a comprehensive **Settings Dialog** managed by a global `SettingsStore`. The dialog allows users to configure application behavior, appearance (dynamic theming), and manage data.

## Tech Stack
- **State Management**: Native Angular Signals (`SettingsService`) with LocalStorage persistence.
- **UI Components**: PrimeNG `Dialog`, `SelectButton`, `InputSwitch` (or `ToggleSwitch`), `Slider`.
- **Theming**: Dynamic CSS Variable injection for Primary Color updates.
