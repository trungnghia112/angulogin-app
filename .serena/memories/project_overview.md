# Project Overview

## Purpose
Chrome Profile Manager - An Angular web application for managing Chrome browser profiles.

## Tech Stack
- **Framework**: Angular v21.1.0 (latest standalone components)
- **Language**: TypeScript 5.9.2
- **Styling**: Tailwind CSS v4.1.12 + PostCSS
- **State Management**: Angular Signals
- **Package Manager**: npm 11.6.2

## Project Status
This is a **fresh scaffold** with the default Angular welcome page. No custom features implemented yet.

## Key Architectural Decisions
- Standalone components (no NgModules)
- Signal-based state management
- External template/style files (no inline)
- OnPush change detection strategy
