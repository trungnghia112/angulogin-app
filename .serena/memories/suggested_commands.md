# Suggested Commands

## Development
```bash
npm start          # Start dev server at localhost:4200
npm run build      # Production build to dist/
npm run watch      # Build with watch mode
npm test           # Run unit tests
```

## Git Commands
```bash
git add -A && git commit -m "feat: description"  # Semantic commits
git push origin main
```

## Angular CLI
```bash
ng generate component path/to/component-name  # Generate component
ng generate service services/service-name     # Generate service
ng serve --port 4201                          # Custom port
```

## System (macOS/Darwin)
```bash
ls -la          # List files
find . -name "*.ts"   # Find files
grep -r "pattern" src/  # Search in files
```
